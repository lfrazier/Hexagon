Creates an SPArtist from the given opaque sp_artist struct.

This convenience method creates an SPArtist object if one doesn't exist, or returns a cached SPArtist if one already exists for the given struct.

Warning
This method must be called on the libSpotify thread. See the "Threading" section of the library's readme for more information.
Parameters
anArtistThe sp_artist struct to create an SPArtist for.
aSessionThe session to create the artist in.
Returns
Returns the created SPArtist object.

Creates an SPArtist from the given Spotify artist URL.

This convenience method creates an SPArtist object if one doesn't exist, or returns a cached SPArtist if one already exists for the given URL.

Warning
If you pass in an invalid artist URL (i.e., any URL not starting spotify:artist:, this method will return nil.
Parameters
aURLThe artist URL to create an SPArtist for.
aSessionThe session to create the artist in.
blockThe block to be called with the created SPArtist object, or nil if given an invalid artist URL.

- (sp_artist *) artist
readnonatomicassign

Returns the opaque structure used by the C LibSpotify API.

Warning
This method must be called on the libSpotify thread. See the "Threading" section of the library's readme for more information.
This should only be used if you plan to directly use the C LibSpotify API. The behaviour of CocoaLibSpotify is undefined if you use the C API directly on items that have CocoaLibSpotify objects associated with them.
