inSession: (SPSession *)  aSession
type: (sp_artistbrowse_type)  browseMode 

Creates an SPArtistBrowse from the given SPArtist.

This convenience method is simply returns a new, autoreleased SPArtistBrowse object. No caching is performed.

Warning
It is strongly recommended that you don't use SP_ARTISTBROWSE_FULL as the browse type, as this will download metadata for every single album and track from the given artist, which is both a lengthy and memory-intensive operation. Please use SP_ARTISTBROWSE_NO_TRACKS (which downloads artist information, their albums and top tracks) or SP_ARTISTBROWSE_NO_ALBUMS (which only downloads artist information and top tracks) instead.
If you pass in an invalid artist URL (i.e., any URL not starting spotify:artist:, this method will return nil.
Parameters
anArtistThe SPArtist to make an SPArtistBrowse for.
aSessionThe SPSession the browse should exist in.
browseModeThe type of artist browse to perform.
Returns
Returns the created SPArtistBrowse object.
