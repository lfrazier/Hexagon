inSession: (SPSession *)  aSession 

Creates an SPAlbum from the given opaque sp_album struct.

This convenience method creates an SPAlbum object if one doesn't exist, or returns a cached SPAlbum if one already exists for the given struct.

Warning
This method must be called on the libSpotify thread. See the "Threading" section of the library's readme for more information.
Parameters
anAlbumThe sp_album struct to create an SPAlbum for.
aSessionThe SPSession the album should exist in.
Returns
Returns the created SPAlbum object.

Creates an SPAlbum from the given Spotify album URL.

This convenience method creates an SPAlbum object if one doesn't exist, or returns a cached SPAlbum if one already exists for the given URL.

Warning
If you pass in an invalid album URL (i.e., any URL not starting spotify:album:, this method will return nil.
Parameters
aURLThe album URL to create an SPAlbum for.
aSessionThe SPSession the album should exist in.
blockBlock to be called with the created SPAlbum object, or nil if given an invalid album URL.

- (sp_album *) album
readnonatomicassign

Returns the opaque structure used by the C LibSpotify API.

Warning
This method must be called on the libSpotify thread. See the "Threading" section of the library's readme for more information.
This should only be used if you plan to directly use the C LibSpotify API. The behaviour of CocoaLibSpotify is undefined if you use the C API directly on items that have CocoaLibSpotify objects associated with them.

readnonatomicstrong

readnonatomicassign

Returns YES if the album is available in the logged-in user's region.

- (SPImage *) cover
readnonatomicstrong

- (SPImage *) largestAvailableCover

- (sp_albumtype) type
readnonatomicassign

Possible values:

SP_ALBUMTYPE_SINGLE Single.

SP_ALBUMTYPE_COMPILATION Compilation.

SP_ALBUMTYPE_UNKNOWN Unknown type.
