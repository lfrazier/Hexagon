Detailed Description

This class performs a search on the Spotify catalogue available to the given session, returning tracks, albums and artists.

Method Documentation

- (BOOL) addAlbumPage

Request an extra page of albums from the Spotify search service.

The albums property will be updated when new results are returned.

Warning
If you want to request more than just albums, use -addPageForArtists:albums:tracks:playlists: for better performance.
Returns
Returns YES if a search request was created, NO if there are no more results to fetch or if a search is already in progress.

+ (SPSearch *) liveSearchWithSearchQuery: (NSString *)  searchQuery
inSession: (SPSession *)  aSession 

Creates a new "live search" SPSearch with the default page size for the given query.

Live searches should be used when displaying "live search" UI — for example, a search suggestion UI as the user is typing into a search field. This convenience method is simply returns a new, autoreleased SPSearch object. No caching is performed.

Warning
The search query will be sent to the Spotify search service immediately. Be sure you want to perform the search before creating the object!

- (NSArray *) artists

- (BOOL) hasExhaustedArtistResults
readnonatomicassign

Returns YES if the search service has indicated there are no more artist results to find.

- (BOOL) hasExhaustedPlaylistResults
readnonatomicassign

Returns YES if the search service has indicated there are no more playlist results to find.

- (BOOL) hasExhaustedTrackResults
readnonatomicassign

Returns YES if the search service has indicated there are no more track results to find.

- (BOOL) loaded
readnonatomicassign

Returns NO if a search is currently in progress, or YES if the search is complete (or has failed).

- (NSArray *) playlists
readnonatomicstrong

Returns the playlist results of the search, or nil if the search has not loaded or there are no playlist results.

- (NSError *) searchError
readnonatomiccopy

Returns an NSError indicating the search failure reason, or nil if there was no error.

- (NSString *) searchQuery
readnonatomiccopy

Returns the search query for this search.

- (sp_search_type) searchType
readnonatomicassign

Returns the type of search, either SP_SEARCH_STANDARD or SP_SEARCH_SUGGEST.

- (SPSession *) session
readnonatomicstrong

Returns the session the search is being performed in.

- (NSURL *) spotifyURL
